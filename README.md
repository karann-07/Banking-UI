# 🏦 Banking Web Application with AI Chatbot

A full-stack banking web application built using HTML, CSS, JavaScript, and Node.js.  
The project features a modern responsive banking UI along with an integrated AI-powered chatbot (BankMate AI) for banking-related assistance.

---

## ⚠️ Disclaimer
This is not an official banking website.  
This project is developed strictly for educational and portfolio purposes.

---

## 🚀 Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla)

### Backend
- Node.js
- Express.js
- Axios
- OpenRouter AI API

---

## 🤖 AI Chatbot – BankMate AI

Integrated a secure AI chatbot that:
- Answers banking-related queries
- Provides information about accounts and loans
- Assists with internet banking guidance
- Responds in short and simple professional messages

Security implemented using:
- Environment variables (.env)
- API key protection
- .gitignore configuration

## 🛠️ How To Run Locally

### 1️⃣ Clone the Repository
```
git clone https://github.com/karann-07/Banking-UI.git
```

### 2️⃣ Navigate to Project Folder
```
cd Banking-Homepage-UI
```

### 3️⃣ Setup Backend
```
cd backend
npm install
```

### 4️⃣ Create .env File Inside backend Folder
```
OPENROUTER_API_KEY=your_api_key_here
```

### 5️⃣ Start Server
```
node server.js
```

### 6️⃣ Open Frontend
Open `index.html` in your browser.
